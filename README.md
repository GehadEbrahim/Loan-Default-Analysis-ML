# Loan Default Prediction & Credit Risk Analysis

## 📌 Project Overview
This project aims to analyze banking data to predict the likelihood of loan defaults. By understanding customer behavior and financial indicators, we can help financial institutions mitigate credit risks.

## 🛠️ Current Progress (In Progress)
- [x] **Data Exploration (EDA):** Understanding features and target distribution.
- [x] **Data Visualization:** Using Seaborn & Matplotlib to find correlations.
- [x] **Data Cleaning:** Handling null values and data inconsistencies.
- [x] **Feature Engineering:** Encoding categorical variables (Label Encoding).
- [ ] **Model Building:** (Next Step) Training Machine Learning classifiers.
- [ ] **Evaluation:** Testing model accuracy and performance.
